
# ✈️ Airline Passenger Data Analysis Project

## Project Title

**Airline Passenger Data Analysis and Visualization Using Python**

---

## 📌 Project Overview

This project focuses on analyzing an airline passenger dataset using Python and popular data analysis libraries such as Pandas, NumPy, and Matplotlib. The notebook performs data exploration, cleaning, statistical analysis, and visualization to gain insights into passenger demographics, flight activity, and airport operations.

The analysis helps understand trends related to passenger ages, gender distribution, flight schedules, nationalities, airport traffic, and flight status.

---

## 🎯 Objectives

* Load and explore the airline dataset.
* Perform data cleaning and preprocessing.
* Analyze passenger demographic information.
* Examine flight activity over weekdays and months.
* Visualize important trends using charts.
* Generate statistical insights from the dataset.

---

## 📂 Dataset Information

The dataset contains airline passenger and flight information, including:

* Passenger ID
* Gender
* Age
* Nationality
* Airport Name
* Country Information
* Departure Date
* Flight Status
* Pilot Name
* Arrival Airport

The dataset is loaded from a CSV file and analyzed using Python.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## 📊 Analysis Performed

### 1. Data Exploration

* Displayed dataset shape.
* Checked data types.
* Detected missing values.
* Previewed dataset records.

### 2. Statistical Analysis

* Calculated average passenger age.
* Generated descriptive statistics for age.
* Counted total passengers.
* Analyzed gender distribution.

### 3. Date and Time Processing

* Converted departure dates into datetime format.
* Extracted:

  * Month
  * Day
  * Weekday
* Analyzed flight frequency by weekday.

### 4. Nationality and Airport Analysis

* Displayed top passenger nationalities.
* Identified busiest airports.
* Examined flight status distribution.

### 5. Data Visualization

The notebook includes several visualizations such as:

* Bar chart for male vs female passengers.
* Pie chart for gender distribution.
* Bar chart for number of flights per weekday.

---

## 📈 Sample Insights

* Passenger age distribution was analyzed using descriptive statistics.
* Flight activity trends were identified across weekdays.
* Gender distribution was visualized clearly using bar and pie charts.
* Airport usage frequency helped identify busy airports.
